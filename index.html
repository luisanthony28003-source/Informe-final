Proyecto de Métodos Numéricos

Aplicación web interactiva para analizar el comportamiento dinámico de un circuito RC mediante métodos numéricos e información experimental obtenida mediante Arduino.

Fundamento Teórico

Un circuito RC está formado por una resistencia y un capacitor, cuyo comportamiento temporal está descrito mediante una ecuación diferencial de primer orden.

Métodos Numéricos Implementados

1. Interpolación de Newton de segundo orden

2. Newton-Raphson

3. Derivación numérica

4. Integración numérica

5. Runge-Kutta de cuarto orden
